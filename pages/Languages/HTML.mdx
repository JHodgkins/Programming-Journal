# HTML
## Overview
HTML means Hyper Text Markup Language, it is the backbone of the webpages we view today, HTML is a markup language used to structure pages.

## Headings
There are six levels of headings
```html
<h1>Heading 1</h1>
<h2>Heading 1</h2>
<h3>Heading 1</h3>
<h4>Heading 1</h4>
<h5>Heading 1</h5>
<h6>Heading 1</h6>
```
